Doenet

Doenet hopes to fully be ready to use in five years. This deadline is set far out on purpose, as the project team intends to complete the project before then. Right now, the project is in its early stages of development, with basic building of the admin, editor, and course functionality underway. The first deadline is by January 15th, when the project is presented in Boulder, CO.

Project Timeline

  • Doenet Works Offline

    November 15th 2019

    Doenet will be able to work fully offline for the ease of students and instructors.

  • Multiple Person Interactions

    December 16th 2019

    More than one person will eb able to interact with the Doenet tool at a time and multiple windows can be open and active.

  • Present at Boulder, CO

    January 15th 2020

  • Begin Creating Course Content

    February 15th 2020

  • Open up to Other Courses

    March 25th 2020

    Begin opening up Doenet to use for other courses, mainly for testing purposes

  • Use Testing From Other Courses For New Features

  • Begin the process towards making Doenet Open Source
